How to use workshop.bioconductor.org

1) Register/Login
2) Find tutorial and Execute
3) Launch RStudio
4) Run tutorial
5) Stop a tutorial

2) Find tutorial and Execute

1 then 2 in screenshot

3) Launch RStudio

Click "Click here to display" (might take a couple of minutes to appear)

4) Run tutorial

For all tutorials except Carpentries, an Rmd will be in the Editor pane. Learners can run the code in this or create a new script/Rmd (File > New File > R Script) to write the code from scratch. For Carpentries, the idea is to start from scratch, so create a new script and type the code. The code to follow along is under "Materials" for each tutorial on this webpage.

5) Stop tutorial

(This will delete it, so download any R code/data you want to keep in RStudio first*)

1 then 2 then 3 in screenshot

Note

  • Only 2 tutorials (per person) can run at the same time.
  • Stop a tutorial before launching another.
  • *Download data you want to keep (such as .Rhistory) from RStudio before stopping, see below.
