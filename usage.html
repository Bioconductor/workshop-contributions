How to use workshop.bioconductor.org

1) Register/Login
2) Find tutorial and Launch
3) Open RStudio
4) Run tutorial
5) Stop tutorial
Note

Remember!

  • DO NOT share your workshop URL (link) on any public forum (eg chat or social media), including in screenshots.
  • Only 2 tutorials (per person) can run at the same time. Stop a tutorial before launching another.
  • Running tutorials will be automatically stopped after 6 days.
  • *Download data you want to keep (such as .Rhistory) from RStudio before stopping, see detailed instructions below.

1) Register/Login

If you have not done so already, navigate to the Bioconductor Workshop Galaxy, and fill out the appropriate form to login or register.

2) Find tutorial and Launch

You may browse available tutorials on the event page.

Once you have picked a tutorial, please find it by searching for its name, or scrolling through the Workshops Panel on the left of your screen.

Launch a tutorial by clicking on its name (1), followed by the Launch button (2), as shown below.

3) Open RStudio

Once you have clicked Launch, you should expect a little wait, followed by a message that will update to include a clickable text once your RStudio has successfully launched.

4) Run tutorial

For most tutorials, except Carpentries, an Rmd will be in the Editor pane. Learners can run the code in this or create a new script/Rmd (File > New File > R Script) to write the code from scratch. For Carpentries tutorials, the idea is to start from scratch, so create a new script and type the code. The code to follow along can be found on the tutorial website. For example, see under "Materials" for each tutorial on the Smorgasbord tutorials webpage .

5) Stop tutorial

(This will delete it, so download any R code/data you want to keep in RStudio first*)

If you are looking to get a certificate, be sure to export your .Rhistory before stopping! Click below for details.

Detailed Instructions for Certificates

In order to stop a tutorial, click the User menu, followed by the Active Workshops, where you will then have the ability to select and stop all active workshops.
